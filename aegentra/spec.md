- **Primary Color**: Deep Green (#0A6F4E) - call-to-action buttons, highlights, icons, brand accents, and section headings
- **Secondary Color**: Soft Mint (#A8E6CF) - backgrounds for cards, sections, and accent elements
- **Text Colors**: Dark charcoal (#2C2C2C) for primary text, ensuring high contrast and maximum readability
- **Secondary Text**: Medium gray (#4A4A4A) for supporting text elements
- **Background**: Clean light backgrounds (#FFFFFF) with subtle texture overlays for modern aesthetic
- **Typography**: Modern tech-focused sans serif with refined hierarchy, balanced sizes and weights for premium enterprise feel
- **Layout**: Clean, spacious design with increased whitespace, modern section spacing, and refined visual rhythm
- **Visual Consistency**: Strict two-color green system (Deep Green + Soft Mint) consistently applied across all pages including header, footer, and hero sections
- **Hero Sections**: High-contrast dark text over light backgrounds with subtle green gradient accents
- **Components**: Unified cards, banners, and forms under the cohesive green-based system with enterprise-grade UI standards
- **Image Integration**: Clean integration of all existing images with proper spacing and visual hierarchy
- **Shadows**: Refined, subtle shadows for depth and modern UI quality
- **Animations**: Smooth micro-animations, transitions, and hover effects for enhanced user interaction
